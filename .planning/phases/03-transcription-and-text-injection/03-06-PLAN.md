  <name>Task 2: Human verification of Phase 3 end-to-end flow</name>
  <action>
With the app running (`python -m linux_speech_flow`), test these scenarios in order and confirm each works:

**Scenario 1 — Basic transcription (TRANS-01, 02, 04)**
Open a text editor (gedit or similar). Click in the text area. Press F9, speak clearly: "Hello world this is a test". Press Escape to stop. Wait for processing (processing.wav then success.wav). Confirm transcribed and cleaned text appears. Check text is not prefixed with "Here is" — LLM should return only the text.

**Scenario 2 — Filler word removal (TRANS-02)**
Click in text editor. Press F9, speak: "um so like uh I wanted to say hello". Press Escape. Confirm pasted text has fillers removed.

**Scenario 3 — LLM fallback (TRANS-05)**
Open Settings, set LLM model to "invalid-model-name". Save. Record and stop a phrase. Confirm raw transcript appears with notification "LLM failed — raw transcript pasted". Restore correct model in Settings.

**Scenario 4 — Terminal paste (TRANS-04)**
Open a terminal emulator. Click in terminal. Press F9, speak a short phrase, press Escape. Confirm text appears (uses Ctrl+Shift+V internally).

**Scenario 5 — Transcription settings UI**
Open Settings. Confirm Transcription section is visible after Audio section with: LLM model field, pipeline timeout, processing/success sound toggles, app category editors, Advanced expander with prompt editor and Reset to Default button. Save without errors.

**Scenario 6 — Sound sequence (TRANS-10)**
Record and stop normally. Confirm: stop chime → processing.wav → success.wav in sequence.

**Scenario 7 — F10 reprocess (TRANS-07, 08)**
Temporarily set a wrong API key. Record and stop. Wait for "Transcription failed — WAV saved" notification. Restore correct API key. Press F10. Confirm single failed WAV retries and pastes. Confirm `~/.local/share/linux-speech-flow/failed/` is empty after success.

**Scenario 8 — Recording queue (TRANS-11)**
Press F9 and speak. While pipeline is running (processing.wav audible), press F9 again immediately. Confirm "Recording queued" notification appears. Wait for both to complete and paste in sequence.
