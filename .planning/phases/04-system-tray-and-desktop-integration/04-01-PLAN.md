  <action>
    Create `src/linux_speech_flow/tray.py` implementing `TrayManager` per the research patterns. Use the minimal skeleton from 04-RESEARCH.md as the base.

    Module-level constants:
    ```python
    RECORDING_FRAMES = ['freeflow-recording-1', 'freeflow-recording-2', 'freeflow-recording-3']
    PROCESSING_FRAMES = ['freeflow-processing-1', 'freeflow-processing-2', 'freeflow-processing-3']
    RECORDING_INTERVAL_MS = 500
    PROCESSING_INTERVAL_MS = 400

    ICON_NAMES = [
        'freeflow-idle',
        'freeflow-recording-1', 'freeflow-recording-2', 'freeflow-recording-3',
        'freeflow-processing-1', 'freeflow-processing-2', 'freeflow-processing-3',
        'freeflow-error',
    ]
    ```

    `install_icons()` standalone function:
    - dest_dir = Path.home() / '.local' / 'share' / 'icons' / 'hicolor' / 'scalable' / 'status'
    - dest_dir.mkdir(parents=True, exist_ok=True)
    - For each name in ICON_NAMES: use `importlib.resources.files('linux_speech_flow.icons').joinpath(f'{name}.svg')` and `importlib.resources.as_file(ref)` context manager. Only copy if dest file does not exist (allows future icon updates — omit the `if not dest.exists()` check actually: always overwrite so updates propagate). Use `shutil.copy2(src, dest)`.
    - Log each icon installed at DEBUG level.

    `TrayManager.__init__(self, app, on_settings, on_debug_log, on_reprocess)`:
    - Call `dbus.mainloop.glib.DBusGMainLoop(set_as_default=True)` at the start (pitfall 1 from research).
    - Build `self._reprocess_item` dict (held reference for mutation):
      `{'type': 'item', 'label': 'Reprocess Failed (0)', 'callback': on_reprocess, 'enabled': False, 'visible': True}`
    - Instantiate `self._tray = TrayIcon(app_id='com.github.linux-speech-flow', title='FreeFlow', icon_name='freeflow-idle')`
    - Set `self._tray.menu_items` to flat list (no separators per CONTEXT.md):
      Settings / Open Debug Log / Reprocess Failed (0) / Quit
      where Quit callback is `app.quit`.
    - Set left-click to open settings: `self._tray.set_left_click(on_settings)` — per research override: left=Settings, right=menu (CONTEXT.md secondary-activate doesn't work with trayer SNI protocol; research established left-click opens Settings as better UX).
    - Animation state: `self._anim_source_id = None`, `self._anim_frames = []`, `self._anim_frame_idx = 0`.

    `setup(self)`: calls `self._tray.setup()`. Must be called from App.do_startup() BEFORE app.run() enters main loop.

    `set_state(self, state: str)`:
    - 'recording' → `_start_animation(RECORDING_FRAMES, RECORDING_INTERVAL_MS)`
    - 'processing' → `_start_animation(PROCESSING_FRAMES, PROCESSING_INTERVAL_MS)`
    - 'idle' → `_stop_animation(); self._tray.change_icon('freeflow-idle')`
    - 'error' → `_stop_animation(); self._tray.change_icon('freeflow-error')`

    `update_failed_count(self, count: int) -> bool`:
    - Mutate dict FIRST (pitfall 5): `self._reprocess_item['label'] = f'Reprocess Failed ({count})'`
    - `self._reprocess_item['enabled'] = count > 0`
    - `self._tray.update_menu()`
    - `return False` (GLib.idle_add convention — pitfall 4).

    `_start_animation(self, frames, interval_ms)`:
    - Always call `_stop_animation()` first (pitfall 3).
    - Set `self._anim_frames = frames`, `self._anim_frame_idx = 0`.
    - Call `self._tray.change_icon(frames[0])` immediately.
    - `self._anim_source_id = GLib.timeout_add(interval_ms, self._advance_frame)`.

    `_stop_animation(self)`:
    - If `self._anim_source_id is not None`: `GLib.source_remove(self._anim_source_id); self._anim_source_id = None`.

    `_advance_frame(self) -> bool`:
    - `self._anim_frame_idx = (self._anim_frame_idx + 1) % len(self._anim_frames)`
    - `self._tray.change_icon(self._anim_frames[self._anim_frame_idx])`
    - `return True` (keep timer running — pitfall of returning None would stop animation).

    Imports needed: `logging`, `importlib.resources`, `shutil`, `pathlib.Path`, `dbus`, `dbus.mainloop.glib`, `gi.repository.GLib`, `trayer.TrayIcon`.
