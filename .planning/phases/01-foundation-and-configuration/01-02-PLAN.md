  <behavior>
    **load_config() behavior:**
    - No config file → returns DEFAULT_CONFIG copy (all fields present)
    - Config file exists → merges file contents over defaults (missing keys get defaults)
    - Returns dict with keys: groq_api_key (str), microphone (str), vocabulary (list[str]), setup_complete (bool)

    **save_config(config: dict) behavior:**
    - Creates ~/.config/linux-speech-flow/ directory if missing (parents=True, exist_ok=True)
    - Writes config as pretty JSON (indent=2)
    - Sets file permissions to 0600 via os.chmod
    - Returns None

    **Test cases:**
    - load_config() with no file → returns dict with "setup_complete": False, "vocabulary": []
    - save_config(config) then load_config() → round-trip equality
    - After save_config, file permissions are 0o600 (oct(stat.S_IMODE(os.stat(path).st_mode)) == '0o600')
    - load_config() with partial config file → missing keys filled with defaults
    - vocabulary saves and loads as list of strings: ["kubernetes", "OAuth"]
    - setup_complete saves True and loads as True (not truthy string)
  </behavior>
  <implementation>
