  <action>
    Create `src/linux_speech_flow/wizard.py` with a `WizardWindow(Gtk.ApplicationWindow)` class.

    **Use Gtk.Stack — NOT Gtk.Assistant** (deprecated since GTK 4.10, removed in GTK 5 — RESEARCH.md pitfall #1).

    **Structure:**
    ```python
    class WizardWindow(Gtk.ApplicationWindow):
        PAGES = ["api_key", "microphone", "vocabulary"]
    ```

    **Page 1 — API Key:**
    - Title: "Step 1 of 3: Groq API Key"
    - `Gtk.PasswordEntry(show_peek_icon=True)` for the key field (built-in masking — no manual toggle)
    - "Validate" button (or Next triggers validation)
    - `Gtk.Spinner` that starts during validation
    - Error label (red text) for inline errors — two distinct messages:
      - "Invalid API key" (401)
      - "Could not connect to Groq — check your internet connection" (ConnectionError/Timeout)
    - Next button disabled until validation passes (`self._api_key_valid = False` flag)
    - Validation runs in `threading.Thread(target=..., daemon=True)`; UI updates via `GLib.idle_add(callback)` where callback must `return False`

    **Page 2 — Microphone:**
    - Title: "Step 2 of 3: Microphone"
    - `Gtk.ComboBoxText` populated from `list_microphones()`
    - If `pulsectl.PulseError` raised: show error label "Could not enumerate audio devices. Is PulseAudio/PipeWire running?"
    - Pre-select mic matching `config["microphone"]` if set (for Settings re-use)
    - Call `list_microphones()` on page entry (not at wizard init) — supports re-enumeration (CONF-05)

    **Page 3 — Vocabulary:**
    - Title: "Step 3 of 3: Vocabulary (optional)"
    - `Gtk.ScrolledWindow` containing `Gtk.TextView` — one word/phrase per line
    - Label: "Enter custom words or phrases, one per line. Leave empty to skip."
    - Pre-populated from `config["vocabulary"]` joined by newlines (for Settings re-use)

    **Navigation:**
    - Back/Next buttons at bottom; Next becomes "Finish" on last page
    - Back disabled on first page
    - On Finish: collect all values, call `save_config()` with `setup_complete=True`, then `self.close()`
    - On window close (X button or cancel): do NOT set setup_complete=True — wizard reopens next launch

    **Vocabulary parsing on Finish:**
    - Split TextView content by newlines → strip each line → filter empty → `list[str]`

    **Pre-loading existing config:**
    - Wizard accepts optional `config: dict` constructor argument (defaults to `load_config()`)
    - If `config["groq_api_key"]` non-empty: pre-fill API key field and mark `_api_key_valid=True`
    - If `config["microphone"]` non-empty: pre-select matching ComboBoxText entry
    - If `config["vocabulary"]` non-empty: pre-populate TextView
  </action>

    **app.py — startup sequence and tray stub:**
    Update `do_activate()` to implement the startup sequence:

    ```python
    def do_activate(self):
        config = load_config()
        if not config.get("setup_complete", False):
            # First run or incomplete setup — show wizard
            if not hasattr(self, "_wizard") or self._wizard is None:
                self._wizard = WizardWindow(application=self)
                self._wizard.connect("close-request", self._on_wizard_closed)
            self._wizard.present()
        else:
            # Setup complete — tray would start here (Phase 4)
            # For now: show a minimal placeholder window so app doesn't exit
            if not hasattr(self, "_placeholder") or self._placeholder is None:
                self._placeholder = self._build_placeholder_window()
            self._placeholder.present()

    def _on_wizard_closed(self, _window):
        self._wizard = None

    def _build_placeholder_window(self):
        win = Gtk.ApplicationWindow(application=self, title="linux-speech-flow")
        menu_bar_label = Gtk.Label(label="Setup complete. Tray will appear in Phase 4.")
        # Add Settings menu item
        btn = Gtk.Button(label="Open Settings")
        btn.connect("clicked", self._on_open_settings)
        box = Gtk.Box(orientation=Gtk.Orientation.VERTICAL, spacing=12)
        box.append(menu_bar_label)
        box.append(btn)
        win.set_child(box)
        win.set_default_size(300, 150)
        return win

    def _on_open_settings(self, _btn):
        settings = SettingsWindow(application=self)
        settings.present()
    ```
