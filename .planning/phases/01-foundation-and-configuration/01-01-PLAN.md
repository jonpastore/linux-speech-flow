  <action>
    Create `src/linux_speech_flow/audio.py` with a single public function:

    ```python
    import pulsectl

    def list_microphones() -> list[dict]:
        """Return available microphone sources, excluding monitor (loopback) sources.

        Returns list of dicts: [{"name": str, "description": str}, ...]
        "name" is the PulseAudio source name (used as config value).
        "description" is the human-readable label (shown in UI).

        Raises pulsectl.PulseError if PulseAudio/PipeWire is unreachable
        (e.g., pipewire-pulse not installed). Callers must handle this.
        """
        with pulsectl.Pulse("linux-speech-flow") as pulse:
            sources = pulse.source_list()
        return [
            {"name": s.name, "description": s.description}
            for s in sources
            if not s.name.endswith(".monitor")
        ]
    ```

    Key decisions:
    - Filter by `name.endswith(".monitor")` — removes PulseAudio loopback monitor sources from mic list
    - Context manager `with pulsectl.Pulse(...) as pulse` — connection is opened/closed per call; safe for re-enumeration on Settings open (CONF-05)
    - Raises `pulsectl.PulseError` on connection failure — wizard and settings callers wrap in try/except with user-visible error message
    - No fallback to subprocess pactl — pulsectl structured API is the locked decision (RESEARCH.md)
  </action>
  <verify>
    ```bash

    .venv/bin/python -c "from linux_speech_flow.audio import list_microphones; print('OK')"
    ```
