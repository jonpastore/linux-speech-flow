**Window class: HistoryWindow(Gtk.ApplicationWindow)**

Constructor signature: `__init__(self, application, history_store)`
- Title: "linux-speech-flow — History"
- Load saved size from config: `config.get('history_window_width', 700)` and `config.get('history_window_height', 500)`, call `set_default_size(w, h)`
- Connect `close-request` signal to `_on_close_request` handler that saves size back to config via load_config/save_config
- Window layout (vertical Gtk.Box):
  - Header bar area: Gtk.Box (horizontal) with window label "Transcription History" on left, "Clear All History" button on right
  - Scrolled area: Gtk.ScrolledWindow (vexpand=True) containing self._listbox (Gtk.ListBox)
  - Footer: Gtk.Box (horizontal, margin 8px) with "Clear Temp Audio Files" button on left
- Set listbox selection_mode to NONE (rows handle their own click)
- Connect listbox `row-activated` signal: `lambda lb, row: row.toggle_expand()`
- Set `listbox.set_activate_on_single_click(True)`

On init, call `self._load_rows()` to populate from HistoryStore.

`_load_rows(self)`: Remove all existing children from listbox. Call `self._history_store.fetch_all()`. If empty, show empty state: Gtk.Label centered with text "No transcriptions yet. Press [hotkey] to start recording." where hotkey is `load_config().get('hotkey_record', 'F9')`. Otherwise, for each sqlite3.Row in result, append `HistoryRow(entry)` to listbox.

`prepend_entry(self, entry: dict) -> bool`: Called via GLib.idle_add from the pipeline thread after a new transcription. Remove the empty-state label if present (check if listbox has a Label child). Create a HistoryRow from entry dict (not sqlite3.Row — HistoryRow must handle both dict and sqlite3.Row access via `entry['key']`). Prepend to listbox. Return False (GLib.idle_add contract).

`_on_clear_history(self, _btn)`: Show confirmation dialog (Gtk.Window, modal=True, transient_for=self) with message "Clear all transcription history? This cannot be undone." and Cancel / Clear buttons. On Clear: call `self._history_store.clear_all()`, then `self._load_rows()`.

`_on_clear_temp_audio(self, _btn)`: Delete all .wav files in `~/.local/share/linux-speech-flow/failed/` using `Path.glob('*.wav')` and unlink each. Show notification "Temp audio files cleared." via Gtk.Label in footer or send_notification.

`_on_close_request(self, _window) -> bool`: Call `w, h = self.get_default_size()`, load config, set `config['history_window_width'] = w`, `config['history_window_height'] = h`, save config. Return False.

**Row class: HistoryRow(Gtk.ListBoxRow)**

Constructor: `__init__(self, entry)` where entry can be sqlite3.Row or dict (use `entry['key']` notation — both support it).

Internal state: `self._expanded = False`

Layout: Outer Gtk.Box (VERTICAL). Contains:
1. `compact_box` (Gtk.Box HORIZONTAL, margins 12/12/8/8, spacing=8, always visible):
   - timestamp label: format `entry['created_at']` as local time string. Parse ISO8601 with `datetime.fromisoformat(entry['created_at'])`, display as `strftime('%b %d %H:%M')` (e.g. "Feb 21 14:30"). xalign=0.
   - duration label: `f"{float(entry['duration_sec'] or 0):.1f}s"` with css class "dim-label". Fixed width approx.
   - preview label: `(entry['processed_text'] or entry['raw_text'] or '')[:80]` truncated, hexpand=True, xalign=0, ellipsize=Pango.EllipsizeMode.END
2. `self._detail` (Gtk.Box VERTICAL, margins 12/12/0/12, spacing=8, visible=False):
   - Section label "Processed" (bold, xalign=0) + Gtk.Label for full processed_text (xalign=0, wrap=True, selectable=True) + Copy button for processed text
   - Separator (Gtk.Separator HORIZONTAL)
   - Section label "Raw transcript" (with dim css class, xalign=0) + Gtk.Label for raw_text (xalign=0, wrap=True, selectable=True) with slightly muted background (add css class "history-raw-bg" — apply inline CSS via Gtk.CssProvider on the label's parent box: `background-color: alpha(@window_bg_color, 0.5)`)
   - Copy button for raw text
   - Context row: Gtk.Label with `f"App: {entry['app_name'] or '—'}  |  Window: {entry['window_title'] or '—'}"`, dim-label class, xalign=0

Copy buttons: `Gtk.Button(label="Copy")` connected to `lambda _btn, t=text: Gdk.Display.get_default().get_clipboard().set(t)`

`toggle_expand(self)`: Flip `self._expanded`, call `self._detail.set_visible(self._expanded)`.
