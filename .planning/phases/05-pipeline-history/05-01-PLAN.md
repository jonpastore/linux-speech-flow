DB path constant:

```python
DB_PATH = Path.home() / ".local" / "share" / "linux-speech-flow" / "history.db"
```

SQLite schema (extensible for Phase 6):
```sql
CREATE TABLE IF NOT EXISTS history (
    id             INTEGER PRIMARY KEY AUTOINCREMENT,
    entry_type     TEXT NOT NULL DEFAULT 'transcription',
    created_at     TEXT NOT NULL,
    duration_sec   REAL,
    raw_text       TEXT,
    processed_text TEXT,
    app_name       TEXT,
    window_title   TEXT,
    extra_json     TEXT
)
```
Plus a DESC index on created_at.

HistoryStore methods:
- `__init__(self, db_path: Path = DB_PATH)` — stores path, calls ensure_schema()
- `_connect(self) -> sqlite3.Connection` — creates parent dirs, connects, sets WAL mode, sets row_factory = sqlite3.Row, returns connection
- `ensure_schema(self)` — creates table and index via context manager
- `insert(self, entry: dict, max_entries: int = 20) -> int` — inserts row with fields (entry_type, created_at, duration_sec, raw_text, processed_text, app_name, window_title), then prunes using the safe subquery form: `DELETE FROM history WHERE id NOT IN (SELECT id FROM history ORDER BY created_at DESC LIMIT ?)` — do NOT use `DELETE ... ORDER BY ... LIMIT` directly (SQLITE_ENABLE_UPDATE_DELETE_LIMIT may not be compiled in). Returns the inserted row id.
- `fetch_all(self) -> list` — SELECT * ORDER BY created_at DESC, returns list of sqlite3.Row
- `clear_all(self)` — DELETE FROM history + VACUUM

Threading contract: HistoryStore creates a new connection per call (_connect). Never shares a connection across calls. This makes it safe to call from any thread.

Imports needed: sqlite3, from pathlib import Path
