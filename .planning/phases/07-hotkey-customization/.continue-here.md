**Full code review completed — 85% confidence. 6 bugs found:**

1. **CRITICAL: `leaked_f9_count` regression** — Ctrl+Alt+R doesn't produce literal text
   in target windows, but transcription.py still sets `leaked_f9_count = 2 if stop_was_f9 else 1`.
   injector.py uses this to send `leaked * 4` backspaces to vim before pasting.
   Result: every vim recording deletes 4-8 real characters. Must set to 0.

2. **`stop_was_f9` rename incomplete** — `transcription.py:132` and `app.py:149` still
   use `stop_was_f9` param name. hotkey.py was already renamed to `_stop_was_hotkey`.
   Not a runtime bug (True/False passes through), but misleading.

3. **Stale notification string** — `transcription.py:355` says "Press F10 to reprocess"
   but hotkey is now Ctrl+Alt+P.

4. **Stale comment** — `app.py:199` says "Called from HotkeyManager when F10 is pressed"

5. **Dead config keys** — `config.py:90-91` still has `conv_hotkey_start: "f11"` and
   `conv_hotkey_feedback: "f12"`. These are never read; hotkeys are now hardcoded.

6. **`tempfile.mktemp()` deprecated** — `transcription.py:169` in `submit_batch_to_file`.
   TOCTOU race. Should use `tempfile.mkstemp()`.
