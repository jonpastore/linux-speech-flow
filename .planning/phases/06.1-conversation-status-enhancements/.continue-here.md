<decisions_made>

- ESC is a normal stop (cancel=False), not a discard — audio is kept and transcribed.
  Test was initially wrong; corrected to match actual behavior.
- try/except in _conv_start resets state before returning False — ensures atomicity
  regardless of what the callback does.
- test_hotkey.py uses class-level @patch decorators with immediate idle_add side_effect
  to make state transitions observable synchronously without a GTK main loop.
- test_injector.py patches subprocess.Popen and subprocess.run separately since
  xclip uses Popen (intentionally no wait) while xdotool uses run.
</decisions_made>
