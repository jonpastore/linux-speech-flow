## Decisions Made

- on_silence_tick receives raw silence_frames (int) not seconds — the manager layer converts to seconds using CHUNK_DURATION, keeping the recorder concern-free of display logic
- Debounce sentinel last_emitted_silence = -1 ensures the very first tick (silence_frames=0, voice detected) always fires rather than being suppressed
- New public methods placed before _update_elapsed to visually group the public API at the top of the method block
