Add HotkeyManager to `App` in `app.py`. Follow the GTK Application lifecycle:
- `do_startup()` runs only in the primary instance (single-instance via D-Bus) — the correct place for HotkeyManager.
- `do_startup()` is called before `do_activate()` and before `app.run()` blocks.
- HotkeyManager.start() launches the pynput Listener daemon thread.
- `GLib.idle_add(self._hotkey_manager.mark_started)` defers the startup guard to the first GTK main loop tick.

**Add to imports at top of app.py:**

```python
from gi.repository import Gtk, Gio, GLib
from linux_speech_flow.hotkey import HotkeyManager
```
(GLib is not imported yet — add it alongside Gtk, Gio)

**Add `_hotkey_manager = None` to `__init__`:**
```python
self._hotkey_manager = None
```

**Add `do_startup()` method** between `__init__` and `do_activate()`:
```python
def do_startup(self):
    super().do_startup()
    self._hotkey_manager = HotkeyManager(
        on_recording_complete=self._on_recording_complete,
        on_recording_error=self._on_recording_error,
    )
    self._hotkey_manager.start()
    GLib.idle_add(self._hotkey_manager.mark_started)
```

**Add recording callbacks** (Phase 2 stubs — Phase 3 replaces with transcription):
```python
def _on_recording_complete(self, wav_path: str) -> None:
    """Called when recording finishes. Phase 3 will transcribe here.
    Phase 2: just log the path so we can verify the WAV was produced.
    """
    print(f"[Phase 2] Recording complete: {wav_path}")

def _on_recording_error(self, message: str) -> None:
    """Called when mic is unavailable. Notification already sent by HotkeyManager."""
    print(f"[Phase 2] Recording error: {message}")
```

**Add `do_shutdown()` method** to cleanly stop the listener on app exit:
```python
def do_shutdown(self):
    if self._hotkey_manager:
        self._hotkey_manager.stop()
    super().do_shutdown()
```
