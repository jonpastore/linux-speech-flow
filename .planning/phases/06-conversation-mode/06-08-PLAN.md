**Step 1: Tray badge icon**

**Step 6: Q&A round (if Groq key configured)**
Type an answer in the text field and click "Submit Answer". Verify:
- "Thinking…" status shown during API call

Click "Done, use current output". If confidence < 95%, verify warning modal appears. Confirm. Verify:
- ConversationQAWindow closes
