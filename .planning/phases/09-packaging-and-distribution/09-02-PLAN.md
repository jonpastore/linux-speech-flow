    ```yaml
      with:
        fetch-depth: 0
    ```
    The `publish` job uses `actions/download-artifact`, so no checkout step there — skip it.

    **Change 2 — In the build job, replace the tomllib version extraction block** with setuptools-scm:
    Replace:
    ```python
    VERSION=$(python3 -c "
    import tomllib
    with open('pyproject.toml', 'rb') as f:
        print(tomllib.load(f)['project']['version'])
    ")
    ```
    With:
    ```bash
    pip install setuptools-scm
    VERSION=$(python -m setuptools_scm)
    DEB_VERSION=$(echo "$VERSION" | sed 's/\.dev.*//' | sed 's/+.*//')
    ```
    Then in the fpm command, replace `-v "$VERSION"` with `-v "$DEB_VERSION"`.

    **Change 3 — Pin PyGObject<3.51 in TWO places in the build job**:
    - The standalone `pip install PyGObject` (build-time dep for cairocffi etc.) → change to `pip install "PyGObject<3.51"`

    **Change 4 — Add --after-install and --after-remove to the fpm command**, and fix the description:
    Add after the last `--depends` line:
    ```bash
    --after-install scripts/postinst.sh \

    ```
    Replace the `--description` value with the locked text:
